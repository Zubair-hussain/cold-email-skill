# Cold Email Generator Skill

## Purpose
Generate cold emails that get replies (not meetings).

## When to Use
Use when:
- Writing outreach emails to leads
- Analyzing a website and converting it into a cold email
- Generating follow-ups

## Core Rules
- No HTML emails
- No buttons or links in first email
- Keep under 120 words
- Only ONE problem per email
- No feature selling
- No long introductions

## Output Requirements
Every email MUST include:
1. Personal observation
2. Clear problem
3. Real-world consequence
4. Soft CTA (question-based)

## Goal
Start conversations — not sell services.

## Behavior
- Think like a conversion expert
- Focus on missed opportunities
- Avoid generic phrases
