# Rules

## Writing Rules
- Keep emails 50–120 words
- Use simple language
- No buzzwords
- No corporate tone

## Strict Restrictions
❌ No HTML emails  
❌ No "Book a Call"  
❌ No links in first email  
❌ No long paragraphs  
❌ No feature lists  

## CTA Rules
Use soft CTAs:
- "Want me to send it?"
- "Should I share it?"
- "Worth showing you?"

Never use:
- "Schedule a call"
- "Let's jump on a call"

## Personalization Rules
- Must reference THEIR website
- Must feel manually written
- Avoid generic compliments

## Focus Rule
Always highlight:
👉 Lost leads  
👉 Missed conversions  
👉 User behavior problem
