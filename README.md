# 🚀 AI Cold Email Generator (Claude Skill)

Generate high-converting cold emails that actually get replies — not ignored.

Built for freelancers, agencies, and developers who want to turn leads into conversations using AI + proven outreach frameworks.

---

## 🔥 Why This Exists

Most cold emails fail because they:
- Sound like templates
- Focus on services (not problems)
- Ask for meetings too early

This system fixes that.

---

## ⚡ What This Skill Does

- Generates **personalized cold emails**
- Uses **real conversion psychology**
- Follows **plain-text, high-reply frameworks**
- Avoids spam triggers (HTML, links, buttons)

---

## 🧠 Core Framework

Every email follows:

1. Observation  
2. Problem  
3. Consequence  
4. Soft CTA  

---

## 📧 Example Output

Hi John,

I tried using your site like a buyer — couldn’t actually browse listings.

Most visitors hit that and go back to Zillow.

If you're getting even 20–30 visitors/day, that’s lost opportunities.

Want me to send a quick breakdown?

– Zubair
